## DocuxmlEditor Guide

## RefStar Guide

### Environement

> Express + React + MongoDB

### Start the MongoDB database

> sudo service mongod start

### Run the server

> $ cd RefStar <br>
> $ npm install <br>
> $ npm run build <br>
> $ npm run start

### Check by visiting a website

> Go to the browser and type: http://localhost:3000/temp
